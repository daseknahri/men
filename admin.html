Admin Security

Update the admin login used for this restaurant. Credential changes are stored in persisted server storage and survive redeploys.

Minimum 3 characters. This becomes the next admin login for this restaurant.
Minimum 8 characters. Leave blank only if you want to keep the current password.

What happens on save

  • Older admin sessions are closed automatically after credentials change.
  • If the instance still relies on env credentials, saving here creates the local persisted auth file for this restaurant.
  • Credential changes are stored at the configured persisted auth path, not only inside the running container.

Handoff policy

Do not deliver the website while default credentials remain active. Confirm the warning disappears in Launch Readiness and test a fresh login before handoff.
